# outlook

This repository includes an **Executive Personal Assistant (Exec PA) kit** you can copy into ChatGPT, Codex, or another LLM to help manage a chaotic schedule, high-volume email, calendar load, and document review workflows.

## What is included

- `exec-pa.md`: A ready-to-use Exec PA operating prompt.
- Structured workflows for:
  - Daily planning and schedule recovery
  - Daily and on-the-fly email summaries
  - Urgency/importance-based triage
  - Continuous calendar evaluation
  - Meeting prep and follow-through
  - OneDrive/document review and report-backs
  - Sales forecast and spreadsheet checks

## Quick start

1. Open `exec-pa.md`.
2. Copy it into your AI assistant as a system or first message.
3. Fill in the placeholders in the **Owner Profile** section.
4. Start each day by asking: `Run my Morning Brief.`

## If you want it to "run this for me"

Because most assistants do not have direct live access to your mailbox/calendar/OneDrive by default, use this pattern:

1. Ask: `Run this for me now.`
2. Paste a **Current State Pack** with:
   - Today's meetings
   - New/urgent emails
   - Top open tasks
   - Any document or spreadsheet excerpts
3. The assistant returns a prioritized action plan, response drafts, and next 30-minute execution block.

## Suggested commands

- `Run this for me now.`
- `Run my Morning Brief.`
- `Run Daily Email Summary.`
- `Summarize these emails now: ...`
- `Evaluate this week of calendar events: ...`
- `Read and report back on this document: ...`
- `Review this sales forecast data: ...`
